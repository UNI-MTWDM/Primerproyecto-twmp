Primer Proyecto

Lista de actividades

  • Crear una aplicación web que muestre una lista de tareas pendientes utilizando HTML, CSS y Responsive Design para el diseño y la estructura de la página. ok
  • Utilizar Flexbox y CSS Grid para crear un diseño adaptable y moderno. ok
  • Utilizar Git y GitHub para el control de versiones del código y permitir la colaboración de varios desarrolladores. ok
  • Utilizar los fundamentos de JavaScript, eventos y arreglos para crear una lista de tareas interactiva y dinámica. ok
  • Utilizar objetos y clases para organizar y estructurar el código de la aplicación. ok
  • Seleccionar y manipular elementos del DOM utilizando JavaScript para actualizar la lista de tareas en tiempo real. ok
  • Utilizar promesas asíncronas, Fetch y LocalStorage para almacenar y recuperar los datos de la lista de tareas de forma persistente. ok
